## 概览

本指南列出了可以触发不同 KYC 结果的文件类型与精确样本。

<Warning>
  请使用 Cloudinary 链接提交这些文件。在支持链接上传的字段中，直接粘贴 URL。
</Warning>

<Note>
  你可以在请求中使用任意其他个人数据。此处结果主要由下方对应的文件样本决定。
</Note>

## 使用方法

1. 选择你要测试的状态。
2. 使用该状态对应的文件类型和 Cloudinary 链接。
3. 复制下方代码块中的 URL，在 KYC 上传流程中作为文件来源链接提交。
4. 按你的常规流程完成验证提交。

## 测试建议

- 文件类型要与上传字段严格对应。

- 如果页面支持文件上传和链接上传，请优先使用链接上传并粘贴 Cloudinary URL。
