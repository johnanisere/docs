Bullring Finance 是一个全球支付平台，帮助企业在多种货币中发送和接收付款，打破国际贸易的壁垒。无论您是向其他国家的供应商付款、接收国际客户的付款，还是管理全球薪资，我们的 API 都提供安全高效的解决方案。

本文档将指导您使用我们的 API，帮助您创建账户、管理多币种付款和处理跨境交易。
