We support a wide range of fiat and digital currencies across multiple payment rails.

### Fiat Currencies

| Currency | Code | Named Account* | Notes |
|----------|------|----------------|-------|
| **USD** | United States Dollar | ✓ | ACH, Wire, SWIFT |
| **EUR** | Euro | | SEPA |
| **GBP** | British Pound | ✓ | FPS |
| **AED** | UAE Dirham | ✓ | UAEFTS |
| **NGN** | Nigerian Naira | ✓ | NIBBS |
| **GHS** | Ghana Cedi | | Mobile Money |
| **ZMW** | Zambian Kwacha | | Mobile Money |
| **CNY** | Chinese Yuan | | Bank Transfer |
| **BRL** | Brazilian Real | | PIX |

<Note>
**Named Account**: Currencies with ✓ provide dedicated virtual account numbers (IBAN, sort code, or local equivalent) in your name for streamlined collections.

**Mobile Money**: GHS and ZMW operate via mobile money networks; no bank account required.

**Offramp only**: CNY and BRL support payouts and stablecoin-to-fiat conversion, but do not support deposits or balance holding.
</Note>

<Note>
**Settlement Times**
- **SEPA**: Instant
- **ACH**: 1 working day
- **Wire**: 1 hour
- **SWIFT**: 2-5 working days
- **Mobile Money**: Instant
- **PIX**: Instant
- **UAEFTS**: Instant
- **NIBBS**: Instant
</Note>
